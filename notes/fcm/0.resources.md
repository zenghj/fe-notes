## FCM 服务器协议

https://firebase.google.com/docs/cloud-messaging/server
目前 FCM 提供以下原始服务器协议：

* FCM HTTP v1 API
* 旧版 HTTP 协议
* 旧版 XMPP 协议

**如果您需要从设备到服务器的上行消息传递功能，则需实现 XMPP 协议。**
